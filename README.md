# truongquochuy777

Portfolio "Luxury Premium Exclusive" cho Trương Quốc Huy

Hướng dẫn nhanh:

- Mở file `index.html` trong trình duyệt để xem giao diện tĩnh.
- Hoặc dùng server tĩnh: `npx http-server -c-1` hoặc `python3 -m http.server` rồi truy cập http://localhost:8080

Thiết kế: sang trọng, glassmorphism, dark/light, responsive, typography cao cấp.
